# 線上考場使用者操作手冊

本手冊說明如何使用線上考場、錯題庫、錯題總複習，以及學習紀錄備份與還原功能。

## 1. 進入首頁

開啟 `web/index.html` 或網站首頁後，會看到所有考場入口。

首頁主要區塊：

- 試題彙整：多年度整理題庫。
- 分年試卷：依年度分類的線上測驗。
- 試題列印：可列印或另存 PDF 的工具。
- 錯題總複習：集中查看所有尚未出庫的錯題。
- 備份與還原：備份或還原學習紀錄。

可以使用搜尋框輸入年度、科目、級別或關鍵字快速找到試卷。

## 2. 進入線上測驗

在首頁找到要練習的試卷，點選「進入線上考場」。

進入後可使用：

- 左側導覽：切換章節或直接進入線上試卷。
- 開始計時：啟動測驗倒數。
- 手機接續：產生 QR Code 或連結，把目前這份試卷的作答進度同步到手機。
- 切換深色：切換深色 / 淺色模式。

## 3. 作答方式

### 單選題

直接點選一個選項。

點選後會立即顯示：

- 是否答對。
- 正確答案。
- 題目解析。

如果答錯，該題會自動加入錯題庫。

### 複選題

先點選所有認為正確的選項，再點「確認選項並看解析」。

確認後會顯示：

- 是否答對。
- 正確答案。
- 題目解析。

如果答錯，該題會自動加入錯題庫。

### 手寫 / 非選擇題

在文字框輸入答案，可自行對照參考答案與評分標準。

目前手寫題不會自動加入錯題庫。

## 4. 送出答案與評分

完成作答後，點選頁面下方「送出答案與評分」。

系統會顯示：

- 總分。
- 答對題數。
- 評語。
- 各題正確答案與解析。

注意：

- 已評分的試卷會鎖定答案。
- 如果要重新開始，可點「清除所有紀錄」。
- 「清除所有紀錄」會刪除該份試卷的作答紀錄、計時與錯題庫。

## 5. 錯題庫

每一份線上測驗都有自己的錯題庫。

### 什麼情況會進入錯題庫

- 單選題答錯。
- 複選題答錯。

目前手寫題不會進入錯題庫。

### 如何進入錯題重練

在試卷頁的測驗區會看到「錯題庫」面板。

如果有錯題，點選「錯題重練」。

進入後只會顯示目前錯題庫中的題目。

### 錯題如何自動出庫

在單一試卷的錯題重練中：

- 第一次答對：保留在錯題庫，連續答對次數變成 1 / 2。
- 第二次連續答對：自動從錯題庫移除。
- 中途答錯：連續答對次數歸零。

## 6. 錯題總複習

首頁有一個獨立區塊「錯題總複習」。

點選「開啟錯題總複習」後，會看到所有試卷中尚未出庫的錯題。

錯題總複習會顯示：

- 來源試卷。
- 題號。
- 題目。
- 選項。
- 正確答案。
- 解析。

### 錯題總複習和單一試卷錯題重練的差異

單一試卷錯題重練：

- 連續答對 2 次會自動出庫。

錯題總複習：

- 不會因為答對 2 次自動出庫。
- 由學習者自行判斷是否已掌握。
- 確認學會後，點「我學會了」才會出庫。

### 如何手動出庫

在錯題總複習頁中，找到已經掌握的題目，點選「我學會了」。

系統會把該題從原本試卷的錯題庫移除。

## 7. 手機接續作答

每一份試卷右上角有「手機接續」。

操作方式：

1. 在目前裝置打開某份試卷。
2. 作答到一半時，點「手機接續」。
3. 系統會產生 QR Code 與連結。
4. 用手機相機或 LINE 掃描 QR Code。
5. 手機會載入目前這份試卷的作答進度、計時與錯題庫。

注意：

- 手機接續是同步「單一試卷」。
- 若要備份所有試卷，請使用首頁的「備份與還原」。

## 8. 備份與還原學習紀錄

首頁右上角有「備份與還原」。

此功能可備份：

- 所有試卷作答紀錄。
- 所有試卷計時資料。
- 所有試卷錯題庫。

### 8.1 下載備份檔

適合電腦使用。

操作方式：

1. 回到首頁。
2. 點「備份與還原」。
3. 點「下載備份檔」。
4. 系統會下載一個 `.json` 檔。
5. 請妥善保存這個檔案。

### 8.2 匯入備份檔

操作方式：

1. 回到首頁。
2. 點「備份與還原」。
3. 點「匯入備份檔」。
4. 選擇之前下載的 `.json` 備份檔。
5. 系統顯示「已還原 X 筆學習紀錄」即完成。

注意：

- 匯入會覆蓋相同試卷 key 的舊紀錄。
- 匯入後重新進入試卷或錯題總複習即可看到資料。

### 8.3 複製備份碼

適合手機與跨裝置使用。

操作方式：

1. 在有學習紀錄的裝置打開首頁。
2. 點「備份與還原」。
3. 點「複製備份碼」。
4. 系統會把一大串備份碼複製到剪貼簿，也會顯示在文字框。
5. 將備份碼傳到另一台裝置，例如 LINE、Email、備忘錄或雲端筆記。

### 8.4 貼上備份碼還原

操作方式：

1. 在要還原的裝置打開首頁。
2. 點「備份與還原」。
3. 把備份碼貼到下方的大文字框。
4. 點「貼上備份碼還原」。
5. 系統顯示「已還原 X 筆學習紀錄」即完成。

注意：

- 備份碼通常很長，請完整複製。
- 少複製任何一段都可能導致還原失敗。
- 建議用 Safari 或 Chrome 開啟，不建議使用 LINE 內建瀏覽器。

## 9. 手機使用建議

手機可以使用：

- 線上作答。
- 錯題重練。
- 錯題總複習。
- 手機接續同步。
- 備份碼備份與還原。
- JSON 備份檔匯入與匯出。

建議：

- iPhone 使用 Safari。
- Android 使用 Chrome。
- 不建議使用 LINE 內建瀏覽器，檔案匯入、下載與 localStorage 可能較不穩定。

## 10. localStorage 資料保存提醒

本系統目前主要使用瀏覽器 localStorage 保存學習紀錄。

資料可能在以下情況消失：

- 使用者清除瀏覽器資料。
- 使用無痕模式。
- 換手機、換電腦或換瀏覽器。
- 瀏覽器或系統清理網站資料。
- 網站網址或網域改變。

建議定期使用「備份與還原」保存學習紀錄。

## 11. 常見問題

### Q1：我換手機後，錯題庫不見了？

localStorage 不會自動跨裝置同步。請在舊手機使用「備份與還原」產生備份碼，再到新手機貼上還原。

### Q2：我清除瀏覽器資料後還能找回嗎？

如果沒有備份，就無法找回。建議定期下載備份檔或複製備份碼。

### Q3：錯題總複習沒有題目？

可能原因：

- 目前沒有錯題。
- 錯題已在原試卷出庫。
- 使用了不同瀏覽器或不同裝置。
- localStorage 被清除。

### Q4：貼上備份碼還原失敗？

請確認：

- 備份碼完整複製。
- 沒有多複製其他文字。
- 使用系統瀏覽器 Safari / Chrome。

### Q5：「我學會了」會影響哪裡？

會從該題原本所屬試卷的錯題庫移除。移除後，該題也不會再出現在錯題總複習。
